Version 2.0 of this shortcode now includes all funcitonality previously provided by the premium Aspen Pro plugin.

== Description ==

This plugin provides a collection of useful shortcodes (including a responsive slider) and widgets compatible with the Aspen Theme. These shortcodes have been selected and developed based on requests and feedback from thousands of users of the Weaver II and Aspen themes.

Includes complete documentation help file.

= Shortcodes included =

* **[aspen_slider]** - This may be the hidden gem in this plugin. The slider is based on the jQuery bxSlider library.
It is a fully responsive slider that can display images, videos, and any HTML content in several different
configurations. All bxSlider options are supported, resulting in one of the most flexible slider plugins for
WordPress available.
* **[aspen_show_posts]** -  Show individual posts based on many options - category, title, and more.
* **[aspen_tab_group]** - Display content in a tabbed box.
* **[aspen_youtube]** - Show your YouTube videos responsively, and with the capability to use any of the YouTube custom display options.
* **[aspen_vimeo]** -  Show your Vimeo videos responsively, and with the capability to use any of the Vimeo custom display options.
* **[aspen_iframe]** - Quick and easy display of content in an iframe.
* **[div]**, **[span]**, **[html]** - Add div, span, and other html to pages/posts without the need to switch to Text view.
* **[aspen_show/hide_if_mobile]** - Show or hide content on mobile devices.
* **[aspen_hide/show_if_logged_in]** - Hide or show content based on user being logged in or not.
* **[aspen_bloginfo]** - Display any information available from WordPress bloginfo function.
* **[aspen_user_can]** - Display content base on logged in user role.
* **[aspen_site_title]** - Display Site title.
* **[aspen_site_desc]** - Display Site tag line.
* **[aspen_extra_menu]** - Show extra menus

= Widgets Included =

* **Aspen 2 Column Text Widget** - Add text into two columns in a widget
* **Aspen Per Page Text Widget** - Add a text widget on a per page basis
* **Aspen Vertical Menu** - Show vertical menus in a widget

== Installation ==

It is easiest to use the Plugins : Add Plugin admin page, but you can do it manually, too:

1. Download the plugin archive and expand it
2. Upload all the plugin files and directories to your wp-content/plugins/aspen-shortcodes-and-widgets directory
3. Go to the Plugins page in your WordPress Administration area and click 'Activate' for this plugin.
